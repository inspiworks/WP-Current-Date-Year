== Description ==
A plugin that gets current year, date, week and month. Just insert the appropriate shortcode and you're good to go. One good application of this plugin would be having a self updating copyright year in the website's footer.

List of shortcodes:

[year] displays the current year (For e.g., 2016)

[month] displays the current month in text format (For e.g., October)

[month format="semi"] displays the first 3 letters of the current month (For e.g., OCT)

[month format="numerical"] displays the numerical value of current month (For e.g., 10)

[week] displays the position of the current week in the complete year (For e.g., 43)

[day reference="year"] displays the position of the current day with respect to the complete year (For e.g., 298)

[day reference="month"] displays the position of the current day with respect to the complete month (For e.g., 24)

[day reference="week" format="numerical"] displays the position of the current day with respect to the complete week (For e.g., 2)

[day reference="week" format="text"] displays the current day in text format (For e.g., Monday)

If you need help or have more questions about this plugin, please [Contact Us](https://inspiworks.com/contact-us)

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-current-date` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place the required shortcode in the page/post/widget area
